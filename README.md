# LipFilter AR
This is a Unity project for Android, where you can test different colors of lipstick using AR.

## Setup
You will need Unity, and an android phone running Android 7.0 or higher.

## Usage
1. Clone this repo, and open it with a Unity editor.
2. In the project folder, open `Assets -> FaceTracking-lipfilter -> CustomFace` scene.
3. you may notice that the canvas is in landscape. deal with it.
4. In the Unity editor, go to `File -> Build Settings`. Select Android and switch platform. You may have to install packages for Android build support.
5. Go to `Player Settings -> XR Plug-in Management`.
6. Select Android and checkmark `ARCore` if not already checked.
